# FitForge — AI Fitness Coach Dashboard

**Train Smarter. Coach Better. Scale Faster.**

A beautiful, production-ready SaaS dashboard designed specifically for fitness coaches and personal trainers. Generate personalized workout plans instantly with AI, manage clients, track progress, and scale your coaching business.

![FitForge Dashboard](https://github.com/yourusername/fitforge/assets/preview-image.webp)

## ✨ Key Features

- **🤖 AI Workout Generator** — Powered by Google Gemini — Create custom workout plans in seconds
- **📊 Modern Analytics Dashboard** — Performance charts, daily stats, and progress tracking
- **👥 Client Management** — Organize and monitor all your clients
- **📅 Today's Workout** — Interactive session tracking with real-time updates
- **🎨 Premium Dark UI** — Neon cyber-fitness design with smooth animations
- **📱 Fully Responsive** — Works perfectly on desktop and mobile

## 🛠 Tech Stack

- **Framework:** Next.js 16 (App Router) + TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Animations:** Framer Motion
- **Charts:** Recharts
- **AI:** Google Gemini API
- **State:** Zustand + localStorage (ready for Supabase)

## Security & Production Ready

- Robust input validation with Zod
- Rate limiting on AI generation
- Protection against prompt injection and API abuse
- Secure architecture patterns
